# 📄 ResumeMatcher AI

An AI-powered Resume Screening and ATS Matching System built using Python and Streamlit.

## 🚀 Features

✅ Upload Resume PDF

✅ Extract Resume Text Automatically

✅ Detect Technical Skills from Resume

✅ Match Resume Against Job Description

✅ Calculate Resume Match Score

✅ Display Matched Skills

✅ Display Missing Skills

✅ AI-Powered Resume Review

✅ Resume Strength Analysis

✅ Resume Weakness Analysis

✅ Skill Gap Identification

✅ Hiring Recommendation

✅ Interactive Resume Match Dashboard

✅ Download Resume Review Report
